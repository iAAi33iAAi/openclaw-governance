## 💬 Sponsorship Philosophy

This project operates on the same principles it encodes:

- **No extraction without reciprocity.** Sponsors receive genuine value — not just a badge.
- **Sovereignty preserved.** Sponsors do not gain ownership, override lineage, or direct architecture.
- **Transparency.** All sponsorship tiers, perks, and uses of funds are published openly here.
- **Love-centered.** Every dollar that flows through this project is validated against the question: *does this increase flourishing, reduce harm, and create beauty?*
