      ## Flow State 1: OpenClaw Governance Pipeline

      ### Sovereignty-First Validation

      Every commit, decision, and deployment flows through the OpenClaw Governance Stack:

      - **Sovereignty Check**  humans remain origin, owner, final authority
      - **Love Quality Validation**  every action must increase flourishing, reduce harm, create beauty
      - **Extraction Prevention**  no action may extract value from humans or biosphere
      - **Consensus Mechanism**  0.67 quorum across 12 primary dimensions required

      **Key files:** `backend/openclaw_flow.py`, `backend/consensus-engine/consensus_algorithms.py`

      ## Quick Start

      ```bash
      # Clone and initialize
      git clone https://github.com/iAAi33iAAi/openclaw-governance
      cd openclaw-governance
      make install

      # Start all services
      make start-all

      # Validate sovereignty
      python backend/openclaw_flow.py

      # Run love quality check
      curl http://localhost:8001/api/love-quality/validate
      ```
